# travel-app-backend

This is a backend repository for [client repository](https://github.com/petrgali/travel-app).

**Technologies:**
- TypeScript
- Nest.js
- Prisma

**Setup:**

Clone this repo and run the following commands:
```bash
  cd travel-app-backend
  # to install packages
  npm install
  # to run website
  npm run start:dev
```
